## What is it?

NeuroAdapt AI is an interactive performance assessment tool that measures how cognitive and motor performance changes under pressure. It was originally inspired by observations of performance instability in sport shooting training — where a well-learned skill can break down when the state of the nervous system changes.

The tool captures four real performance dimensions and generates a personal **NeuroAdapt Performance Profile** with a Clinical Summary Report.

---

## The Four Tests

| Test | What it measures |
|------|-----------------|
| ⚡ Reaction Timing | Neural response latency to visual stimuli |
| 〜 Rhythm Consistency | Temporal pattern maintenance under sustained attention |
| ◈ Accuracy Under Pressure | Decision accuracy with progressive time pressure |
| ▶▶ Speed Under Pressure | Motor performance degradation under maximum load |

---

## Performance Profile — 5 Dimensions

- **Reactivity** — speed of neural response
- **Stability** — consistency across time
- **Pressure Sensitivity** — performance drift under load
- **Cognitive Control** — accuracy under cognitive pressure
- **Cognitive Decline Risk (CDR) Index** — composite pre-clinical marker

## Scoring — Important Note

Current scoring thresholds are based on **general population estimates** and logical construction — not formal clinical normative data. The tool requires validity and reliability studies before any clinical deployment.

Potential calibration directions (under exploration):
- Population norms for visual reaction time by age group
- Executive function benchmarks (ADHD, frontal lobe conditions)
- Correlation with validated batteries (CPT-3, BRIEF-2, MoCA)

---

## Potential Clinical Applications
